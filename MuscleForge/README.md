# 🏋️‍♂️ MuscleForge (MF) – מאגר הכושר המלא

## 🔥 1. מבוא – מהות המאגר

מאגר זה נבנה כדי לייצר בסיס כוח גופני–מנטלי יציב, בשילוב עקרונות נשימה, תנועה ודיוק, מסודר לפי רמות ומקטעים.

---

## 💪 2. אנטומיה בסיסית – קבוצות שרירים

* חזה – עליון / אמצעי / תחתון
* גב – רחב גבי / זוקפים / טרפז
* כתפיים – קדמי / אמצעי / אחורי
* יד קדמית – דו־ראשי
* יד אחורית – תלת־ראשי
* רגליים – קדמית / אחורית / ישבן / שוק
* ליבה – בטן / אלכסוניים / מותן

## 📅 4. תוכנית שבוע ראשון – MF Base Program

### יום 1 – חזה + תלת

* לחיצת חזה 4×8
* דחיקה עליון 3×10
* שק 3×8
* פוש־דאון 3×12
* פלאנק 1 דקה

### יום 2 – גב + דו

* חתירה 4×10
* מתח 3×6–8
* פולאובר 3×12
* כפיפת מרפקים 3×10
* בטן V – 3×15

### יום 3 – רגליים

* סקוואט 4×6–8
* מכרעים 3×12
* כפיפות רגליים 3×10
* הרמות עקב 4×20
* פלאנק צד 2×30ש׳

### יום 4 – כתפיים + ליבה

* לחיצת כתף 4×8
* הרחקה לצדדים 3×12
* הרחקה אחורית 3×15
* סופר־סט בטן

## 🥗 5. תזונה – Fuel Mode

**חלבון:** 1.6–2.2 גר׳ לק"ג גוף
**פחמימות:** 30–50% תפריט
**שומנים:** בריאים בלבד
